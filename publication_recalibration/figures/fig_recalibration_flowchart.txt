================================================================================
                         KEY METHODOLOGY NOTES
================================================================================

1. Keyword weights derived from TRAINING set using log-likelihood ratios
2. Percentile thresholds (P96, P78, P4) calibrated on TRAINING set
3. ALL 4 METHODS tested on held-out TEST SET (n=11,150)
4. Rule-Based selected for: 
   - Highest accuracy (66.4%, tied with Evidence-Based)
   - Greater parsimony (simpler than Evidence-Based)

Severity Classification Priority: Assign highest severity level matched
